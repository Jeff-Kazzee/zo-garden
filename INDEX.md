# The Garden

A hand-curated directory of Skills, Personas, Prompts, tools, writing, and projects worth knowing about — for people building on [Zo Computer](https://zo.computer) and adjacent AI tooling.

Each entry is a link to where the work actually lives. We don't host code here. We point at it, vouch for it, and prune what stops being good.

> Submissions are reviewed individually. "Useful project" is not enough — tell us *why it matters*.
