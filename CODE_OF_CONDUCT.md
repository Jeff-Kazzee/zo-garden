# Code of Conduct

## Our standard

This project is a public space. Treat other contributors like you'd want to be treated. Disagreement is welcome. Hostility, harassment, and discrimination are not.

## Examples of behavior that helps

- Direct, specific feedback on code and submissions.
- Pushback that explains its reasoning.
- Asking questions when you don't understand something.
- Crediting other people's work.

## Examples of behavior that gets you removed

- Personal attacks, insults, or hostile language directed at contributors.
- Harassment of any kind — public or private.
- Discrimination based on race, gender, sexuality, religion, disability, nationality, or any other identity.
- Publishing private information without consent.
- Bad-faith argument: deliberately misreading, sealioning, derailing.
- Spam, self-promotion that doesn't add value, or low-effort submissions.

## Enforcement

Reports go to [@Jeff-Kazzee](https://github.com/Jeff-Kazzee) via GitHub direct message or by opening a private security advisory on this repo.

Consequences range from a comment correction, to a temporary ban, to permanent removal from the project. The maintainer's decision is final.

## Scope

This applies to all project spaces — issues, PRs, discussions, and any communication explicitly tied to the project.
